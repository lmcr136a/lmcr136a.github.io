# Personal Portfolio — File Structure

```
your-github-repo/
│
├── index.html          ← main page
├── style.css           ← all styles
├── script.js           ← scroll animations & nav
│
├── profile.png         ← your profile photo (replace this)
├── cv.pdf              ← your CV (replace this)
│
├── images/
│   ├── prof_exp1.png   ← experience item 1 image
│   ├── prof_exp2.png   ← experience item 2 image
│   ├── prof_exp3.png   ← experience item 3 image
│   ├── ucla-logo.png   ← UCLA logo
│   └── snu-logo.png    ← SNU logo
│
└── projects/
    ├── agrisc.zip         ← project 1 zip
    ├── slam-pipeline.zip  ← project 2 zip
    └── project3.zip       ← project 3 zip
```

## GitHub Pages Setup

1. Push everything to a GitHub repo (e.g. `yourusername.github.io`)
2. Go to **Settings → Pages → Source: main branch / root**
3. Your site will be live at `https://yourusername.github.io`
